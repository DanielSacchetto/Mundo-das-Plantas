O Mundo das Plantas é um viveiro que se dedica na produção de suas palmeiras e se destaca pela grande diversidade de espécies.

São elas: Areca-Bambu (Dypsis lutenscens) ; Areca-de-Locuba (Dypsis madagascariensis) ; Azul (Bismarkia nobilis) ; Butiá-Azedo (Butia capitata) ; Carpentária (Carpentaria acuminata) ; Garrafão (Hyophorbe lagenicaulis) ; Guariroba (Syagrus oleracea) ; Imperial (Roystonea oleracea) ; Jerivá (Syagrus romanzoffiana) ; Latânia (Latania lontaroides) ; Leque (Licuala grandis) ; Licuri (Syagrus coronata) ; Macarturi (Ptychosperma macarthurii) ; Palmito Açaí (Euterpe oleracea) ; Palmito Juçara (Euterpe edulis) ; Pescoço Marrom (Dypsis lastelliana) ; Phoenix (Phoenix roebelenii) ; Rabo de Peixe (Caryota mitis) ; Rabo de Raposa (Wodyetia bifurcata) ; Rafis (Raphis excelsa) ; Seafórtia (Archontophoenix cunninghamii) ; Solitária (Ptchosperma elegans) ; Tamareira (Phoenix dactylifera) ; Tamareira das Canárias (Phoenix canariensis) ; Triangular (Dypsis decaryi) ; Veitchia (Veitchia merrillii) ; Washingtonia (Washingtonia filifera).

Dispomos de técnica especializada para preparo, sangria, descanço, transporte, plantio e cuidado com as palmeiras.

Trabalhamos com a venda de palmeiras jovens e adultas de todo tamanho, realizamos projetos paisagísticos e jardins.
